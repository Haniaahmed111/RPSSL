**Uge 5 – RPSSL**
Dette projekt er et lille spil, hvor man spiller Rock–Paper–Scissors–Spock–Lizard mod en agent. Den første der får 3 point vinder.

**Hvad programmet gør**
- Brugeren vælger en af de 5 figurer.
- Agenten vælger tilfældigt.
- Programmet afgør vinderen med RPSSL-reglerne.
- Point opdateres efter hver runde.
- Spillet stopper automatisk ved 3 point.
